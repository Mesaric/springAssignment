Endpoint Description
/hello-rest{?language=ISO_CODE}> Returns string of 'Hello World'. If optional language parameter is provided, its translation(version) will be returned instead (if in postgresDB).
/hello{?language=ISO_CODE} Returns web page that displays of 'Hello World'.
/secure/hello{?language=ISO_CODE} Returns string of 'Hello World' but requires authentication.
/secure/admin Admin page that allows admin to add new Language-Text pairs in DB. Requires authentication with ADMIN account.
/external/hello{?language=ISO_CODE} Returns string of 'Hello World' translated with external API.
